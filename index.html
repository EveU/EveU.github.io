Portfolio

  • Language Resources

    Ruby on Rails and PostgreSQL form the back-end API, and ReactJS creates the front-end user interface. The project is a resource site for language learners to browse and search for foreign media, such as books and songs, at an appropriate level for their language ability. Future developments would include filtering functionality and authorisation levels for new content submission.

  • Flight/Hotel Search

    A flight and hotel comparison site, built with JavaScript as a team project. The site displays flight and hotel information, and users can search by destination and departure date to receive package deals, ordered by price. Through this project I gained experience collaborating on GitHub, branching off from the main code and merging updates in once signed off by the whole team.

  • Tic Tac Toe

    An online version of Tic-Tac-Toe created in Ruby on Rails, with game logic implemented through models and associations. Authentication and authorisation control user access, for example only allowing the current player to place a piece. Also includes a basic computer player, and a 'Recent Games' table where users can view previous results and return to incomplete games.
